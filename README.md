# Redirector to /four

A simple website that smoothly redirects users to `makka.lol/four` with a loading animation.

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   `npm install`
2. Run the app:
   `npm run dev`

## Deployment

This project is optimized for **Cloudflare Pages**.

- **Build command:** `npm run build`
- **Output directory:** `dist`
